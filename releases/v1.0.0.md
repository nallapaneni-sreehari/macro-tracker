# MacroTracker v1.0.0 — Initial Release

> Know your fuel. Own your gains.

The first public release of MacroTracker — a cross-platform nutrition tracking app built with Ionic Angular and Capacitor, backed by a Node.js + MongoDB server and powered by the USDA FoodData Central database.

---

## What's Included

### 🔐 Authentication
- Passwordless email login via one-time code (OTP)
- No password storage — codes expire after single use
- Session persists across app restarts; data tied to email and synced from server on every login

### 🍽 Meal Logging
- Organize food entries by named meals (Breakfast, Lunch, Dinner, Snacks, or custom)
- Add, edit, and remove individual food items per meal
- Mark meals as complete
- Per-meal macro breakdown with expandable detail view

### 🔍 Food Search
- Search across 500,000+ items from the USDA FoodData Central database
- Full macro breakdown: calories, protein, carbs, fat, fiber per serving
- Adjust serving size inline before logging
- Recent searches cached locally for speed

### 🤖 Milo — AI Nutrition Assistant
- Conversational AI assistant for nutrition queries
- Instant macro estimates for custom/homemade meals
- Meal suggestions based on remaining daily macros
- Persistent chat history per session

### 🎯 Goals
- Set custom daily targets: calories, protein, carbs, fat, fiber
- Goals synced to server and restored after reinstall or login on a new device
- Live progress rings and percentage badges on the Today page

### 📅 History & Calendar
- Calendar date picker to browse any past day
- Full meal log rendered for selected date
- Days with logged data highlighted on the calendar

### 💧 Water Tracker
- Log water intake in 250 ml increments
- Daily water progress shown on Today page
- Negative intake clamped to zero

### 📋 Meal Templates
- Save any meal as a reusable template
- Apply a template to the current day in one tap
- Rename and delete templates

### 🎨 Themes
- **Aurora** — Light purple, modern SaaS
- **Ember** — Warm orange-red, energetic
- **Midnight** — Deep dark, OLED-friendly
- Theme preference persisted locally

### 🔔 Toast Notifications
- Custom toast notification system (no native Ionic toasts)
- Variants: success, error, warning, info, offline
- Glassmorphism card design with colored left stripe, icon pill, and countdown progress bar
- Adapts to active theme (light/dark aware)
- Server connectivity loss detected and surfaced as a persistent offline banner

### 📱 Android
- App ID: `in.iamsreehari.macrotracker`
- Custom app icon generated from SVG source across all mipmap densities
- Custom splash screen for all orientations and densities
- `minSdkVersion 22`, `targetSdkVersion 34`

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Ionic 8 + Angular 20 |
| Native runtime | Capacitor 8 |
| Backend | Node.js + Express |
| Database | MongoDB Atlas |
| AI | OpenAI API |
| Food data | USDA FoodData Central API |
| Auth | Custom OTP via email (Nodemailer) |
| Hosting | `macrostracker.iamsreehari.in` |

---

## Known Limitations (v1.0)

- Android only (iOS build not yet configured)
- No barcode scanner in this release
- Food search requires internet; no offline food database
- OTP codes delivered by email — delivery speed depends on mail provider
